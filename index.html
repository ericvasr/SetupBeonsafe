BeonSafe Setup

Instale o BeonSafe Setup

Execute o seguinte comando em seu terminal:

bash <(curl -sSL setup.beonsafe.com.br/beonsafe_script.sh)

Funcionalidades

Docker: Instalação e configuração do Docker Engine
Portainer: Interface web para gerenciamento de contêineres
Watchtower: Atualização automática de contêineres
Pacotes Essenciais: Ferramentas úteis para administração
